Staff Alert Emails

Manage who gets notified when new referrals arrive. Use the "✉ Alert Staff" button on any referral to send a targeted alert.

ACOE Staff Alert List
Staff on this list appear in the ✉ Alert Staff panel on each referral card. Select who should be notified and the app opens a pre-filled email in your mail client. Alert history is saved on each referral record.

✉ Alert Staff About This Referral

Select staff members to notify, then click Send. This opens a pre-filled email in your mail client. To manage this list, go to the Staff Alerts tab.

⚙ Connect to Google Sheets

One-time setup. Every referral submission will write to your shared Sheet in real time.

Step 1 — Create a Google Sheet

sheets.google.com → blank spreadsheet → name it ACOE Community School Referrals.

Step 2 — Paste this Apps Script

Extensions → Apps Script → delete everything → paste below → replace YOUR_SHEET_ID with the ID from your Sheet URL.

Step 3 — Deploy as Web App

  • Deploy → New deployment → Web app
  • Execute as: Me · Who has access: Anyone
  • Deploy → authorize → copy the URL

⚠ Must be "Anyone" (not "Anyone with Google account"). Create a new deployment after every script edit.
